安装使用教程

PE的制作

材料

找一个不用u盘（以下过程会格式化u盘）

一个大容量硬盘（用于备份文件）

————————————————————————————

首先，进入你pe的官网（这里推荐微pe，一下以微pe制作）

微pe官网：www.wepe.com.cn

然后点击下载

这时候点击最新版本的下载地址

有能力的可以捐赠一下他们，但是没能力的请点先不捐赠。

然后就会出现下载按钮，点一下。

等待下载。这时候可以拿出U盘

下载完成后打开它

然后点击右下角的U盘

调整参数，看不懂的不要只需要选择你的u盘就可以点开始了（此过程会格式化你的u盘，==请记得备份！！！！请记得备份！！！！请记得备份！！！！请记得备份！！！！请记得备份！！！！请记得备份！！！！请记得备份！！！！==)

下载镜像

登录我们的官网www.xiahuaos.top

点击开始使用，然后下载wim文件，建议存储在非系统盘的盘上

现在我们就可以

开始安装

用大容量硬盘备份好你的文件，然后拔出u盘

现在先打开你的PE

每一台电脑打开的方式都不一样

具体见https://www.laomaotao.net/quickquery

现在重启电脑，用上表进入boot

然后使用上下键和enter选择你的u盘

然后等待pe开机

打开CGI备份还原

按要求填写信息

点击执行

等待一会

然后完成后不要关机，打开Dism++修复引导

重启。。。。。

恭喜你，系统安装成功

如果出现这样的蓝屏，按回车(由受害者提供)
